# Deploying this project with GitHub Pages

This repository contains a static site (`index.html`), so you can publish it directly with GitHub Pages.

## Publish from the repo root

1. Open the repository on GitHub.
2. Go to **Settings**.
3. In the sidebar, open **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the **main** branch.
6. Select the **/(root)** folder.
7. Save.

Your site should then be published at:

`https://afhamahmed1.github.io/cnn-playground/`

## Notes

- This site is static, so it does not need a build step.
- Future edits to `index.html` on `main` will automatically update the published site.
